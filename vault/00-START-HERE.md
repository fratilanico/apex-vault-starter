# Welcome to Your APEX Vault

This is your personal AI-powered knowledge base. Claude can read, search, and write to every note in this vault.

## Your 14 Folders

| Folder | Purpose |
|--------|---------|
| **Tasks** | Active work items and to-dos |
| **KnowledgeBriefs** | Things you've learned — auto-captured by AI |
| **Agents** | Notes about AI agents and their roles |
| **Skills** | Reusable patterns and techniques |
| **Decisions** | Architectural and strategic decisions |
| **Incidents** | Problems encountered and how they were fixed |
| **Repos** | Notes about codebases and projects |
| **DailyNotes** | Daily logs |
| **WeeklyReviews** | Weekly summaries |
| **Signals** | Market signals, research, ideas |
| **Governance** | Rules, protocols, standards |
| **Leads** | Business development notes |
| **AgentLogs** | Logs from AI agent sessions |
| **PRs** | Pull request notes and reviews |

## How to Use With Claude

Claude can access this vault via the `apex-vault` MCP tool. Try asking:

- *"Search my vault for anything about [topic]"*
- *"Write a note to my KnowledgeBriefs folder about what we just built"*
- *"What are my 10 most recently modified notes?"*
- *"Read my note called [note name]"*

## Getting Started

1. Start writing notes in any folder
2. Use [[wiki-links]] to connect notes together
3. Ask Claude to write KnowledgeBriefs after important sessions
4. Review your DailyNotes folder to track progress
